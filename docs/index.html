🛒 Curador AVR — Bookmarklet

Botón para tu barra de favoritos del navegador. Lo clickeás cuando estás en una ficha de producto de AliExpress o eBay y captura el producto a tu Supabase con la fórmula de margen aplicada.

Paso 1 — Mostrar la barra de favoritos. Si no la ves, en Chrome: Ver → Mostrar siempre la barra de favoritos, o atajo Cmd/Ctrl+Shift+B.
Paso 2 — Arrastrá este botón a la barra de favoritos:

📌 Curador AVR

Si no se deja arrastrar (algunas combinaciones de browser/sistema no permiten drag de un link javascript:), copiás esto y lo pegás manualmente como URL de un nuevo bookmark: 
javascript:(function(){var s=document.createElement('script');s.src='https://robinsep-cell.github.io/bot-escreaper/curador.js?v='+Date.now();s.async=true;document.body.appendChild(s);})();
Paso 3 — Usalo.
  1. Navegás AliExpress como siempre, encontrás un producto bueno
  2. Seleccionás la variante que querés (color, lado, tamaño)
  3. Click en 📌 Curador AVR en tu barra de favoritos
  4. Aparece un popup con título, precio, envío, variante (lo que la página te muestra a vos)
  5. Editás cualquier campo si hace falta, agregás vehículos compatibles
  6. Click 💾 Guardar → producto entra a Supabase con tu fórmula aplicada
Importante: el bookmarklet es upsert por URL. Si lo apretás dos veces en la misma URL, actualiza el registro (refresh manual). Para refrescar precio/envío de un producto ya curado, simplemente volvés a esa página y apretás el botón otra vez.

¿Qué hace exactamente?

  1. Detecta si es AliExpress o eBay desde el dominio.
  2. Lee del DOM lo que tu navegador ya renderizó: título, imagen, precio en CLP, envío, variante seleccionada.
  3. Muestra todo en un popup editable.
  4. Calcula Costo = (Producto + Envío) × 1.19 y aplica tu fórmula escalonada para sacar PV.
  5. POST a la tabla productos_curados de tu Supabase con upsert por URL.

Privacidad

El bookmarklet solo se ejecuta cuando vos hacés click. No envía nada cuando navegás normalmente. La key de Supabase que usa es la publishable (no service_role), con permisos limitados.
